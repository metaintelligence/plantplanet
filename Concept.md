# PlantPlanet

- K-Startup 초격차 스타트업 프로젝트(DIPS)  
  중기부에서 진행하는 초격차 기술을 수요기업의 현장 인프라 적용으로 혁신적 협업 모델 창출을 지원하는 프로그램


# 당사 지원 과제

- 수요기업: 한국수목원정원관리원
- 과제 내용: 생성형 AI 기반 스토리텔링 식물해설 콘텐츠 생성 및 현장 실증
  - 수목원은 생물자원 DB 및 해설 콘텐츠를 다수 보유하고 있으나, 관람객 특성(연령/관심사/언어)에 따른 맞춤형 콘텐츠 제공 체계 부재.
  - 외국인 방문객 증가, 디지털 기반 관람 수요 확대 및 기후위기/생물다양성 메세지의 효과적 전달을 위한 생성형 AI 기반 체험형 콘텐츠 개발이 요구됨.


# 당사(메타인텔리전스)의 과제 지원 전략
- 생성형 AI 콘텐츠 생성 및 관리 체계를 구축하는 수목원 콘텐츠 플랫폼 개발.
- 한국수목원정원관리원(이하 한수정) DB와 연동된 생성형 AI를 통하여 한수정에 의해 주도적으로 콘텐츠를 생성하고 관리하는 플랫폼 개발 및 콘텐츠 개발 지원.
- 수목원의 환경, 관람객 특성, 콘텐츠의 목적 등 실제 현장의 여건과 의도를 그대로 반영하는 맞춤형 콘텐츠 설정으로 AI 콘텐츠 페이지를 생성하고 디스플레이 함.
- 실제 현장에 맞게 생성된 콘텐츠를 적절한 형태의 기기(키오스크, 모바일QR, 기배치된 단말기기 등)를 유연하게 지원하여 관람객에게 효율적으로 콘텐츠를 전달함.


# 개발 방안


## 플랫폼 형태

---

- 플랫폼은 React+Vite 형태로 개발하여, 다양한 기기 여건에도 공용적으로 지원할 수 있도록 개발함.
- 관리자 포털: 콘텐츠 생성 및 전체 콘텐츠의 관리(CRUD 지원 및 단말 원격 제어 지원).
- 콘텐츠 페이지: 콘텐츠 생성 설정에 의해 AI로 생성된 콘텐츠 페이지.


## 맞춤형 콘텐츠 데이터

---

- 한수정 DB와 연동하여 콘텐츠 페이지 생성 시 데이터를 읽어 생성 시 반영함.
- 수목원 특성을 반영하는 콘텐츠 생성 설정 포맷(json) 개발(하기 '고급 생성 설정' 항목 참조.)


## 콘텐츠 생성

---

- 맞춤형 콘텐츠 데이터를 기반으로 하여 Back-end LLM을 통하여 React 페이지로 생성함.


## 고급 생성 설정

---

- 대상 식물 선택
- 현재 계절 선택(관람시점): 봄/여름/가을/겨울
- 주요 설명 항목 선택(다중 선택 가능)
    - 생김새, 생태, 이름유래, 문화/역사, 활용, 보전가치, 비교학습, 재미요소(냄새, 촉감, 독성, 특이점)
- 배포 용도
    
    
    | 배포 용도 | 설명 |
    | --- | --- |
    | 식물 표찰 QR | 특정 식물 앞에서 바로 접속 |
    | 전시관 키오스크 | 큰 화면용 콘텐츠 |
    | 모바일 관람 코스 | 여러 식물을 연결 |
    | 교육 프로그램 | 수업/체험 활동용 |
    | SNS/홍보 페이지 | 행사 홍보용 요약 콘텐츠 |
- 페이지 템플릿 선택: 식물 소개 페이지, 만화로 보여주는 스토리텔링 페이지, 퀴즈 페이지 등의 유형화된 페이지.
    
    
    | 상호작용 | 설명 |
    | --- | --- |
    | 퀴즈형 | 객관식/OX/단답형 |
    | 미션형 | “이 식물의 잎 모양을 찾아보세요” |
    | 스토리형 | 만화로 보여주는 해당 식물 스토리텔링 |
    | 관찰형 | 꽃/잎/열매/향기 등 관찰 유도 |
    | 소개형 | 단순 정적 소개 |
- 페이지 레이아웃 선택
    - 페이지 레이아웃 설정 시 고정된 레이아웃을 사용하여 페이지가 생성됨
    - 템플릿 별 페이지 레이아웃에 대한 CRUD 지원.
    - 신규 레이아웃 생성 시 AI에 의해 새로운 레이아웃을 가진 React 페이지가 생성됨.

- 페이지 목적 선택
    
    
    | 목적 | 예시 |
    | --- | --- |
    | 일반 해설 | 식물의 특징을 쉽게 소개 |
    | 교육 | 학생·어린이 학습용 |
    | 체험/미션 | 현장에서 참여 유도 |
    | 캠페인 | 생물다양성, 기후위기, 멸종위기 메시지 전달 |
    | 전시 홍보 | 특정 기획전·계절 행사 안내 |
    | 관람 동선 유도 | 다음 장소나 관련 식물로 이동 유도 |
- 페이지 톤 선택
    
    
    | 톤 | 설명 |
    | --- | --- |
    | 친근한 해설사 톤 | 일반 관람객용 |
    | 동화/캐릭터 톤 | *아동용 한정 |
    | 박물관/전시 해설 톤 | 공공기관 느낌 |
    | 감성 스토리텔링 톤 | 사진·계절 콘텐츠와 잘 맞음 |
    | 학술/전문가 톤 | 식물 정보 중심 |
    | 캠페인 톤 | 기후위기·보전 메시지 강조 |
- 대상고객 선택(다중선택): 아동용, 성인용
- 대상언어 선택: 외국인(대상 언어별 지원 가능)
- 실제 현장 선택
    - 실내
        - 온실, 정원
    - 실외
        - 야외 정원, 숲길, 공원
- 추가 기능 옵션 선택
    - 음성 해설
    - 질문답변AI
    - 유사식물카드
    - 스토리형 전용 옵션
        
        
        | 스토리 시나리오 | 설명 |
        | --- | --- |
        | 멸종된다면? | 사라졌을 때 생태계·인간 생활 변화 |
        | 식물의 하루 | 식물을 주인공으로 한 일상형 이야기 |
        | 시간여행 | 과거·현재·미래의 식물 이야기 |
        | 기후위기 생존기 | 기후변화 속 식물의 변화 |
        | 이름의 비밀 | 이름 유래 중심 이야기 |
- 목표 체험 소요 시간(사용자의 페이지 예상 소모 시간): 10초/30초/1분/3분
- 추가 요청사항 직접 입력
    - PLACEHOLDER 예: “초등학생도 이해할 수 있게 해줘”, “기후위기 메시지를 강조해줘”, “퀴즈를 3개 넣어줘”, “감성적인 문장으로 만들어줘”

## 기본 생성 설정

---

고급 설정에서 아래 항목만 사용함

| 순서 | 기본 설정 항목 | 이유 |
| --- | --- | --- |
| 1 | **대상 식물 선택** | 페이지의 주제. 반드시 필요 |
| 2 | **페이지 템플릿 선택** | 결과물의 형태를 결정. 품질 안정화 핵심 |
| 3 | **페이지 목적 선택** | 일반 해설인지 교육인지 캠페인인지 방향 결정 |
| 4 | **대상 고객 선택** | 문장 난이도와 표현 방식 결정 |
| 5 | **대상 언어 선택** | 외국인 대응 과제 요구와 연결 |
| 6 | **현재 계절/관람 시점 선택** | “지금 무엇을 보면 되는지”를 반영 |
| 7 | **목표 체험 소요 시간** | 콘텐츠 길이와 밀도 결정 |

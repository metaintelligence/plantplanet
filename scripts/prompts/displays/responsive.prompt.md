# responsive 레이아웃 규칙

`deploymentUse`가 명확하지 않거나 전용 규칙 파일이 없을 때 사용하는 기본 반응형 규칙입니다.

## 목표

- 부모 컨테이너의 폭에 맞춰 자연스럽게 렌더링됩니다.
- 관리자 프리뷰, 일반 브라우저, 향후 임베드 환경에서도 내용이 잘리지 않습니다.

## 필수 레이아웃 규칙

- 루트 요소에서 `overflow: "hidden"`으로 콘텐츠를 자르지 않습니다.
- 고정 폭 대신 `width: "100%"`, `maxWidth`, 반응형 grid/flex를 사용합니다.
- 텍스트 컨테이너에는 `minWidth: 0`, `boxSizing: "border-box"`를 적용해 넘침을 방지합니다.
- 이미지 크롭이 필요한 경우를 제외하고, 실제 콘텐츠를 숨기는 overflow 처리를 하지 않습니다.
- viewport height에 의존해 전체 콘텐츠 높이를 고정하지 않습니다.

## 자체 점검

- 가로 폭이 줄어도 텍스트가 겹치지 않는지 확인합니다.
- 콘텐츠가 길어지면 자연스럽게 아래로 흐르는지 확인합니다.
